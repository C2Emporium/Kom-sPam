# Kom-sPam
KomèsPam est une application de caisse et de gestion pour petits et moyens commerçants. Elle permet de gérer les ventes, le stock, les crédits clients et les rapports au quotidien, avec une interface simple et rapide, pensée pour être utilisée sur le terrain.
